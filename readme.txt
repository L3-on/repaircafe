Die Website wurde für den Desktop-PC (Windows oder Mac, Viewport: Breite 1280 px) 
entworfen. Sie funktioniert aber auch auf eine sich ändernde Viewport-Breite.
Bei der Erstellung wurde folgende Soft- & Hardware verwendet:

Software:
-Windows 10 Version 21H1 (Build 19043.1586)
-Visual Studio Code (Version 1.61.0)
-Google Chrome (Version 100.0.4896.75) (Offizieller Build) (64-Bit)

Hardware:
-NVIDIA GeForce RTX 2060
-Samsung SSD 860 EVO 1TB
-Intel Core i7-9750H


Bei der Systemvoraussetzung ist sich an den Anforderungen von Chrome zu halten um die 
maximale Funktionalität zu gewährleisten.

Windows:
Wenn Sie den Chrome-Browser unter Windows verwenden möchten, benötigen Sie mindestens:

-Windows 7, Windows 8, Windows 8.1, Windows 10 oder höher
-einen SSE3-fähigen Intel Pentium 4-Prozessor oder höher
Hinweis: Für Server ist Windows Server 2008 R2, Windows Server 2012, 
Windows Server 2012 R2 oder Windows Server 2016 erforderlich.

Mac:
Wenn Sie den Chrome-Browser auf einem Mac verwenden möchten, benötigen Sie mindestens:

-OS X El Capitan 10.11 oder höher


Weitere Funktionalität:

-Für die CSS-Übergänge:
	transition { ... }

wird Chrome ab der Version 26 benötigt.
Außerdem: 4.0 -webkit-

Diese Funktion wurde zur Sicherheit auch für Safari, Mozilla Firefox & Opera optimiert,
genauso wie die Effekte:
	blur()
